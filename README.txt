Фреймворк Flask позволяет разделять большой сложый проект
на набор независимых модулей.
Каждый модуль может содержать свои шаблоны, свои стили оформления,
свои стили изображений, и все это будет храниться в отдельном
подкаталоге, что делает фрагмент приложения совершенно независимым.
При необходимости этот каталог можно скопировать в другой проект
и подключить к другому сайту.
Blueprint - эскиз, чертеж.
